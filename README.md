# drone-core

The core crate for Drone, an Embedded Operating System.

## Documentation

- [Drone Book](https://book.drone-os.com/)
- [API documentation](https://docs.rs/drone-core/0.10.0)

## Usage

Place the following to the Cargo.toml:

```toml
[dependencies]
drone-core = { version = "0.10.0" }
```
